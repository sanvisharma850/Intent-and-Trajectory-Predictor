# Intent-and-Trajectory-Predictor

This repository contains **IntentFormer-3D**, a deep-learning framework for
simultaneously predicting the *intent* and the *multi-modal future trajectory*
of traffic agents from nuScenes sensor data.

See [`IntentFormer-3D/README.md`](IntentFormer-3D/README.md) for full
documentation including setup, training commands, and evaluation results.
